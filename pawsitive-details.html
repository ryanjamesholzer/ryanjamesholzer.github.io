Project information

  • Category: Full stack web design
  • Project date: January, 2023
  • Project URL: https://github.com/ryanjamesholzer/pawsitive-outcome

Project Details

Pawsitive Outcome is a full stack web application I built with a small team in early 2023. It was made to be used by a dog adoption company, in order to manage their adoptions and clientele. It was a fun project for our small team, dubbed the Whisker Watchers, and we learned a lot along the way.
